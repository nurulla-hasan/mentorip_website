# 🛡️ MentorIP

> **Global Intellectual Property Law Firm in Bangladesh**

MentorIP is a premium digital platform for a leading international IP law firm specializing in Trademarks, Patents, Copyrights, and Industrial Designs. This website showcases world-class legal services, empowering brands and protecting innovations globally.

---

## ✨ Features

- **🌐 Global Reach**: Showcases international IP protection services across multiple jurisdictions.
- **🎨 Premium UI/UX**: Crafted with a modern aesthetic, featuring smooth micro-animations and a responsive layout.
- **🔒 Secure Client Intake**: Robust form validation for inquiries and service requests.
- **⚖️ Expert Directory**: Dedicated profiles for our top-tier legal professionals.
- **📱 Fully Responsive**: Seamless experience across mobile, tablet, and desktop devices.

## 🛠️ Tech Stack

This project leverages the latest and most powerful web technologies:

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/) & [Lucide Icons](https://lucide.dev/)
- **Form Handling:** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) (v18+ recommended) and [pnpm](https://pnpm.io/) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory:**
   ```bash
   cd mentorip_website
   ```

3. **Install dependencies:**
   ```bash
   pnpm install
   ```

4. **Start the development server:**
   ```bash
   pnpm dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
